ℹ️ このExtensionについて

おはツイKeeper Chrome Extension v2.0

✨主な機能
  • X(Twitter)の投稿をワンクリックでおはツイ登録
  • 登録済みかどうかの自動確認
  • 連続投稿日数・統計をポップアップで表示
  • 登録解除もポップアップから操作可能
  • API v2 対応
